Cálculo del área en una curva normal

library(tidyverse)

El contenido de este documento sirve para preparar las gráficas que usaremos en la actividad en clase. Los estudiantes usan una calculadora web para obtener el área bajo la curva en una normal. Enlace a la calculadora de la curva normal: http://www.rossmanchance.com/applets/ISI/NormCalc.html

En total, hay seis viñetas para que los estudiantes calculen las áreas. Todas se basan en una normal N(0,1).

Viñeta 1

funcShaded <- function(x) {
  y <- dnorm(x,mean = 0,sd=1)
  y[x < 0.93] <- NA
  return(y)
}

graf1 <- cnormal  + stat_function(fun=funcShaded, geom="area", fill="red") +
scale_x_continuous(breaks = c(-4,-2,0,0.93,2,4)) +  
scale_y_continuous("Densidad") +
theme_bw() +
theme(axis.text.x=element_text(colour="black", size = 14))  
ggsave("./v_aleatoria/graf1.png",
       units = "cm",
       width = 10,
       height = 10
      )
